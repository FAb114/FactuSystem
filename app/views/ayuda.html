Introducción a FactuSystem

Bienvenido a FactuSystem, su solución integral para la facturación y gestión comercial multisucursal. Este manual le guiará a través de todas las funcionalidades del sistema para optimizar la gestión de su negocio.

Características Principales

  • Facturación electrónica integrada con AFIP (ARCA)
  • Gestión de ventas, compras e inventario
  • Control de caja y flujo de efectivo
  • Soporte para múltiples sucursales con sincronización
  • Modo offline/online con sincronización automática
  • Integración con métodos de pago digitales (Mercado Pago, bancos)
  • Sistema de cuotificación para pagos con tarjeta
  • Reportes detallados y exportables

Este manual está diseñado para guiarlo paso a paso en la utilización de todas las funciones disponibles en FactuSystem.

Vista general del Dashboard de FactuSystem

Primeros Pasos

Inicio de Sesión

Para acceder al sistema, debe ingresar sus credenciales en la pantalla de inicio:

  1. Ingrese su nombre de usuario
  2. Ingrese su contraseña
  3. Si tiene habilitada la autenticación de dos factores, ingrese el código de verificación
  4. Haga clic en "Iniciar Sesión"

Pantalla de inicio de sesión

Selección de Sucursal

Si su cuenta tiene acceso a múltiples sucursales, se le solicitará seleccionar la sucursal en la que desea trabajar:

  1. Seleccione la sucursal del menú desplegable
  2. Haga clic en "Continuar"

Apertura de Caja

Antes de comenzar a operar, debe registrar la apertura de caja:

  1. Ingrese el monto inicial en efectivo
  2. Opcionalmente, añada una nota o comentario
  3. Haga clic en "Abrir Caja"

Formulario de apertura de caja

Navegación por el Dashboard

El Dashboard está compuesto por:

  • Sidebar izquierdo: Menú con acceso a todos los módulos
  • Área central: Espacio donde se muestran las pestañas con los módulos abiertos
  • Barra superior: Información de usuario, sucursal y botones de acción rápida

Facturador

El módulo de facturación está diseñado para permitir la emisión de facturas de manera rápida y eficiente.

Creación de una Nueva Factura

Selección del Cliente

En el campo "Nombre del Cliente" puede:

  • Presionar Enter sin datos para seleccionar "Consumidor Final"
  • Presionar F1 para abrir el formulario de datos completos del cliente
  • Comenzar a escribir para buscar clientes existentes

Tipo de Comprobante

Seleccione el tipo de comprobante a emitir:

  • Factura A (Responsable Inscripto)
  • Factura B (Consumidor Final)
  • Factura C (Monotributista)
  • Factura X (Sin validez fiscal)
  • Presupuesto

Ingreso de Productos

En el campo "Código de Barras / Nombre" puede:

  • Escanear un código de barras con un lector
  • Escribir el código manualmente
  • Comenzar a escribir el nombre del producto para usar autocompletado
  • Presionar F1 para abrir el buscador avanzado de productos

Modificación de Productos

  • Presione F2 sobre un producto ya agregado para modificar su cantidad o precio
  • Haga clic en el ícono de eliminar para quitar un producto de la lista

Descuentos y Recargos

Para aplicar descuentos o recargos:

  • Ingrese "-10" para un 10% de descuento
  • Ingrese "+5" para un 5% de recargo

Métodos de Pago

Seleccione uno o varios métodos de pago:

  • Efectivo: Ingrese el monto recibido y el sistema calculará el cambio
  • Tarjeta: Especifique tipo (débito/crédito), banco y número de cuotas
  • Transferencia: Confirme la recepción y el monto
  • QR (Mercado Pago): El sistema detectará automáticamente cuando el pago sea exitoso

Facturación Electrónica

Para emitir la factura electrónica:

  1. Complete todos los datos requeridos
  2. Haga clic en "Facturar con ARCA (AFIP)"
  3. Espere la confirmación del sistema

Finalización

Una vez generada la factura, puede:

  • Imprimir ticket (58mm)
  • Imprimir factura (A4)
  • Enviar por email
  • Enviar por WhatsApp

    ¿Puedo usar FactuSystem en múltiples equipos?

    Sí, FactuSystem está diseñado para funcionar en múltiples equipos conectados a la misma base de datos. Puede configurar tantas terminales como requiera su negocio, todas sincronizadas con un servidor central.

    Para una correcta configuración multiterminal, debe asegurarse de que todas las computadoras estén en la misma red local y tengan acceso al servidor donde se aloja la base de datos principal.

    ¿Qué hago si se corta internet durante la operación?

    FactuSystem está diseñado para funcionar en modo offline. Si se corta la conexión a internet, puede seguir operando normalmente. El sistema almacenará temporalmente los datos en la base local y los sincronizará automáticamente cuando se restablezca la conexión.

    La única limitación durante el modo offline es que no podrá emitir facturas electrónicas AFIP, pero el sistema guardará todos los datos necesarios para emitirlas posteriormente cuando se recupere la conexión.

    ¿Con qué frecuencia debo hacer respaldos?

    FactuSystem cuenta con un sistema de respaldo automático que puede configurarse desde la sección Configuraciones > Backups. Recomendamos:

    • Respaldo diario automático (configurado por defecto)
    • Respaldo manual semanal adicional
    • Respaldo mensual completo que incluya configuraciones del sistema

    También es recomendable almacenar copias de los respaldos en ubicaciones externas (nube o dispositivos externos).

    ¿Cómo emito una factura electrónica?

    Para emitir una factura electrónica, debe:

    1. Crear una nueva factura desde el módulo Facturador
    2. Completar todos los datos requeridos (cliente, productos, forma de pago)
    3. Hacer clic en el botón "Facturar con ARCA (AFIP)"
    4. Esperar la confirmación del sistema

    Es importante haber configurado previamente la integración con ARCA en la sección Configuraciones > Integraciones > ARCA.

    ¿Qué hago si la factura electrónica fue rechazada?

    Si la factura fue rechazada por AFIP, el sistema mostrará un mensaje con el motivo del rechazo. Los casos más comunes son:

    • CUIT inválido o inexistente: Verifique los datos del cliente
    • Error de conexión: Verifique su conexión a internet
    • Certificado vencido: Actualice su certificado en Configuraciones > Integraciones > ARCA

    Una vez solucionado el problema, puede intentar emitir la factura nuevamente.

    ¿Cómo emito una nota de crédito?

    Para emitir una nota de crédito:

    1. Vaya a la sección Documentos > Notas de Crédito
    2. Haga clic en "Nueva Nota de Crédito"
    3. Seleccione la factura asociada (opcional)
    4. Complete los datos requeridos y el motivo
    5. Haga clic en "Emitir Nota de Crédito"

    Si la nota está asociada a una factura electrónica, también debe emitirse electrónicamente.

    Problemas de Sincronización

    Si experimenta problemas de sincronización entre sucursales:

    1. Verifique su conexión a internet

    3. Vaya a Configuraciones > Sincronización y haga clic en "Forzar Sincronización"

    Problemas de Impresión

    Si no puede imprimir facturas o tickets:

    1. Verifique que la impresora esté conectada y encendida
    2. Vaya a Configuraciones > Impresión para revisar la configuración
    3. Pruebe con el botón "Imprimir Página de Prueba"
    Más detalles

    Errores en Facturación AFIP

    Si tiene problemas con la facturación electrónica:

    1. Verifique su conexión a internet
    2. Confirme la validez de su certificado digital
    3. Revise los datos fiscales del cliente
